# SocialHub

SocialHub is a social media web application built using Python, Django, SQLite, and Bootstrap. It allows users to register, log in, create posts, interact with other users, and manage their profiles.

## Features

- User Registration and Login
- User Logout
- Profile Picture Upload
- Update User Bio
- Create and Delete Posts
- Like and Unlike Posts
- Comment on Posts
- Follow and Unfollow Users
- View User Profiles
- View All Registered Users
- Django Admin Panel

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap

## Installation

```bash
git clone https://github.com/amaanvg/SocialHub.git
cd SocialHub
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
